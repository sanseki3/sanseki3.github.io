site pessoal

Eu quero reunir aqui o que vale a pena guardar.

Este site existe para funcionar como arquivo pessoal, banco público de notas e oficina de pensamento. Algumas páginas começam pequenas, outras viram ensaios; todas podem crescer, mudar e ficar mais úteis com o tempo.

Como usar este site: leia como quem entra em uma biblioteca em construção. Nem tudo aqui nasce fechado. Algumas notas são sementes; outras já estão relativamente estáveis. O foco não é publicar volume, mas manter um rastro de pensamento que continue valendo para o meu eu futuro.
