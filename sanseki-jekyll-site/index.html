site pessoal

Um arquivo vivo para pensamento, imaginação e memória.

Este site existe para funcionar como banco público de notas, diário de sonhos, oficina de pensamento e arquivo pessoal de longo prazo. Algumas páginas nascem como semente; outras vão se tornando ensaios, mapas ou registros mais densos com o tempo.

Como usar este site: leia como quem entra em uma biblioteca em construção. Nem tudo aqui nasce pronto. Algumas páginas são abertas, incrementais e deliberadamente inacabadas. O foco é deixar rastros úteis para o meu eu futuro.
